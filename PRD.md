# 내 자기소개 랜딩페이지 PRD

## 1. 목적
오프라인과 온라인에서 업계 모임을 갖거나, 강의 때 만나는 수강생과 교육담당자에게 종이 명함 대신 건넬 자기소개 웹페이지(온라인 명함)를
만든다. 상대방이 qr을 인식하거나 내가 상대방에게 카톡·문자로 링크 한 줄만 보내면 상대방이 바로 열람할 수 있어서,
변경된 내 정보를 갱신하면 최신의 나의 자기소개를 제공할 수 있도록 하는게 목적이다.

## 3. 기능 (필수 / 선택)
### 필수 기능
1. 페이지 맨 위에 왼쪽에 'small N잡러'를 본문 글씨크기로 표시하고, 그 옆에 이름 '한덕환'을 표시(글씨크기 본문의 2배 크기)
2. 다음 줄에 직업 4개(유튜브작가/Ai컨설터트/길여행가/프리랜서강사)(글씨크기 본문과 동일 크기).
3. 다음 줄에 '《이제는,좋아하는 것만 하는 사람》' 를 bold체로 표시.
4. 현직과 전직 경력을 현직 최근 경력 순으로 한 줄 씩 잡 나열 나열.
5. 다음 줄에 '《그전엔.직장인 30년》'을 bold 
6. 다음 줄에 '큰기업근무 15년.작은기업경영 15년' 표시.
7. 끝에 '이메일 보내기' 버튼이 있고, 누르면 기본 메일 앱이 열리며 수신자 주소(hanmokhae@gm가 자동으로 채워진다.
8. 하단에 '전화 걸기' 버튼이 있고, 누르면 전화번호(010-3280-4309)가 클립보드에 복사됩니다.
9. '유튜브 @혼자걷다보면' 링크가 있고, 링크를 누르면 유튜브 주소(youtube.com/@혼자걷다보면)이 자동으로 열린다.
10.'브런치 (투빈대디)' 링크가 있고, 링크를 누르면 브런치 주소(bruch.co.kr/@2beendaddy)가 자동으로 열린다.
11. qr코드(https://m.site.naver.com/25YMN)를 만들어 표시하고, 인식하면 자동으로 열린다.

### 선택 기능 (여유 있을 때)
1. 페이지 상단에 원형 프로필 사진 한 장이 표시된다.
2. 프로필 사진 상부 50%부터 위로 푸른 숲 배경을 삽입한다.
